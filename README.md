# AI Resume Screening System

## Overview
This project is a simple AI-powered resume screening system that evaluates candidates based on a job description.

## Features
- Match score (0–100)
- Candidate ranking
- Key strengths & gaps
- Final recommendation

## Tools Used
- ChatGPT
- Google Sheets

## How It Works
1. Input job description and resumes
2. AI analyzes candidate profiles
3. Outputs structured evaluation
